## FPAA Architecture

<img src="img/1_io_fpaa_architecture.png" width="80%">

The FPAA has 4 Configurable Analog Blocks (CABs). Signal processing is accomplished using an architecture based on switched capacitor circuit design. Every CAB contains two op-amps, a comparator, eight 8-bit capacitors, one SAR and a collection of configurable routing and clock resources.

## IO Cells Types

The AN231E04 doesn't have a single uniform IO pin. Instead, it has 4 "kinds" of IO cells, each belonging to a type that determines the circuitry available.

### Type 1 (and Type 1a)

<img src="img/1_io_cells_type1_1a.png" width="30%">

Most capable IO cells. They contain both:

- Passive elements (R, C, switches)
- Active elements (op-amps, buffers, filters)

Type 1 and 1a IO cells can implement:

- Direct analog input
- Direct analog output
- Active filters (e.g., Sallen-Key type blocks inside the IO cell)
- Sample-and-hold circuits
- Digital input
- Digital output

They also have continuous-time input/output filters whose behavior depends on:

- Internal configuration
- External components (caps you place on the board)

Summary of the available Type 1 and 1a cell options in Anadigm Designer 2.

<img src="img/1_io_summary_type1_1a.png" width="80%">

### Type 2 (and Type 2a)

<img src="img/1_io_type2_2a.png" width="40%">

These are simplified IO cells, meant for cleaner or more "raw" analog access.

They can implement:

- Direct input
- Direct output
- Reference voltage output (VREF, VMR, etc.)
- Digital input/output

They don't include the extra active analog stages that Type 1 cells have, so they behave more like "analog pins" with minimal processing.

Summary of the available Type 2 and 2a cell options in Anadigm Designer 2.

<img src="img/1_io_summary_type_2_2a.png" width="80%">

### Chopper Amplifier

<img src="img/1_io_type1a_2a.png" width="40%">

Only Type 1a and Type 2a can access the chopper amplifier. This amplifier dramatically reduces offset and flicker noise. It is used for very low-level analog signals (microvolt-millivolt range).

!!! info
	Only a single IO cell in a configuration is allowed access to the shared chopper amplifier resources.

<img src="img/1_io_cell_table.png" width="80%">

## IO Voltages

All analog signals inside the FPAA are differential and must be centered around an internal 1.5V reference (VMR).

The AN231/AN241 run only on 0 to 3.3V. There is no negative supply. Therefore, the chip cannot accept or create negative voltages on any analog node.

!!! note
	Signals must always stay between 0V and 3.3V. 
	
	This is unlike an op-amp running +/-12V that can swing below 0V.

Because the FPAA can't swing negative, it creates a virtual analog ground at VMR = 1.5V.

!!! info
	**All analog signals are differential, centered on VMR.**

### Concept of Differential Voltage

Fully differential analog signal processing has the following properties:

- No node ever needs to be negative
- Noise is cancelled between the two rails
- Signals swing around VMR
- Gains, filters, integrators all operate on voltage differences, not absolute voltages

!!! note
	As long as both $V_+$ and $V_-$ of an IO cell stay inside 0V to 3.3V, the FPAA is free to interpret their difference as a signal ranging from -3.3V to 3.3V internally.

!!! Example
	- $V_+ = 1.2V$
	- $V_- = 1.8V$
	
	No voltages are negative. No limits violated.
	
	The CAB interprets: $V_{diff}=V_+ - V_- = -0.6V$

The CAB's circuits (gain cells, multipliers, filters, etc.) always operate differentially, never single-ended. This is identical to how audio differential amplifiers and ADCs work.
